# Arbeidsoppgave 1

Dette er en personlig portefølje bygget med rene webteknologier (HTML, CSS og JavaScript). Nettsiden presenterer min
erfaring, utdanning, prosjekter og hobbyer.

---

### **Teknologier**

Selve porteføljen er laget uten rammeverk, og har derfor ingen avhengigheter. Følgende teknologier er brukt:

- HTML
- CSS
- JavaScript

---

### **Installasjon og kjøring**

1. Klon repoet:
   ```bash
   git clone https://github.com/Friftycode/type-brew.git src/gokstad/work-requirement/01
   ```
2. Ingen avhengigheter trengs for selve porteføljen, siden den er bygget uten rammeverk.
3. Start server:
   ```bash
   npm run dev
   ```
