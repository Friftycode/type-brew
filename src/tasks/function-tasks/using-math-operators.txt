Task 4: Using math operators

Continuing on task 1, we will be using the math operators you created.

Create several functions that will use the operators to solve the following equations.
Print the result and then return it.

1.  a + b - c
2. a / b * c
3. (a - b) * c

Examples:
1.
    - 1,2,3 => 0
    - 4,6,2 => 8
    - 9,0,2 => 7

2.
    - 4,2,3 => 6
    - 10,5,6 => 12
    - 0,100,200 => 0

3.
    - 10,3,2 => 14
    - 8,2,1 => 6
    - 1,0,8 => 8
